# Music Band Rock - Landing Page

## Description
Landing page for a music band. The site presents the band's identity, upcoming concerts, biography, and a photo gallery of their performances.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
```
maqueta-banda/
├── index.html       Main HTML file with all sections
├── style.css        Styles for the entire page
└── script.js        Scroll animation for the hero section
```

## Sections
- Header with logo and navigation
- Hero section with main title and call to action
- Upcoming concerts table
- About the band with image and philosophy list
- Photo gallery
- Footer with contact and social media

## How to Run Locally
1. Clone or download this repository
2. Open the `index.html` file in any modern browser
3. No installation or dependencies required
